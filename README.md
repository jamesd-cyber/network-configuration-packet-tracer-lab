# Network Configuration Project – Cisco Packet Tracer
## Overview
In this lab, I designed and configured a small wired and wireless network using Cisco Packet Tracer. The objective was to simulate a functional small office network including DHCP configuration, static IP addressing, wireless security, MAC filtering, port forwarding, and connectivity testing.
## Key Configurations
- Configured WAN settings with Static IP
- Changed LAN network to 192.168.10.0/24
- Configured DHCP range (192.168.10.20 – 192.168.10.200)
- Assigned static IP addresses to server and staff PC
- Implemented WPA2 wireless security
- Configured MAC address filtering
- Set up port forwarding (HTTP to internal server)
- Tested connectivity using ping
## Validation
Ping tests confirmed successful communication between devices and proper network configuration.
